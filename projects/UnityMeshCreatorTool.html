Unity Mesh Creator Tool

About

Unity tool that is able to create, edit and paint Meshes in real time aswell as save them as a 3D model file.

  • Tool interface window editor that changes on user selection.
  • C++ DLL backend that handles mesh calculations.
  • Edit a mesh by vertex or face.
  • Paint meshes with vertex color.
  • Export finished mesh as a model.
