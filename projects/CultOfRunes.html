Cult of Runes

About

Cult of Runes is a fantasy roguelike first person shooter developed for PC which accomplishes frenetic combat achieved by customizable spells in which you will fight against fearsome enemies in a Norse mythological environment.

An ambicious project in which we had to deploy different systems to handle its scale such as:

  • Custom event system which works through a simple Scriptable Object interface reducing the logic scripts load.
  • Robust and scalable enemy, spell and stat systems.
  • Independant systems to reduce almost entirely scripts dependencies, communication goes one way.
  • Implement various tools in order to reduce the workload of our artists and designers as well as ours.
  • Multiple documents to check, fix and deploy builds after correcting listed bugs and issues.
